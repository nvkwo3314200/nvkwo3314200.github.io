#### 查看内核
```
    cat /etc/redhat-release
```


#### 修复命令(root 权限)
```
    yum update kernel-tools
    yum update kernel-tools-libs
    yum update python-perf
    yum update kernel
    yum update kernel-headers
    
    # 重启
    reboot
```
